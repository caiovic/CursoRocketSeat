## A propriedade background-origin é quem define o ponto de origem de uma imagem específica.

```css
/* Principais valores */
background-origin: border-box; /* a imagem se inicia da borda*/
background-origin: padding-box; /* a imagem se inicia dentro da caixa*/
background-origin: content-box; /* se inicia dentro do conteúdo*/
```

## O background-clip define se a cor ou imagem do background iniciam debaixo de sua área de borda, preenchimento ou conteúdo.

```Css
/* Principais valores */
background-clip: border-box; /* a cor ou imagem de fundo fica atras da linha*/
background-clip: padding-box; /* se inicia a cor dentro da caixa*/
background-clip: content-box; /*a cor se inicia a partir do conteudo */
background-clip: text; /* a partir do texto*/
